

An AI-powered web application that analyzes a user's CV and recommends suitable career paths based on skill matching, skill gap analysis, and learning roadmap generation.

Features
- CV skill extraction using NLP
- Career match score calculation
- Skill coverage analysis
- Missing skill detection
- Learning roadmap with course links
- Interactive charts (Chart.js)

Tech Stack
- Python (Flask)
- Machine Learning / NLP
- HTML, Bootstrap
- JavaScript
- Chart.js
